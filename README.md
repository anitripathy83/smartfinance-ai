# SmartFinance AI — Odoo ERP Module

An AI-powered financial intelligence module built on Odoo 19 for the Odoo Hackathon.

## Features
- 📊 **KPI Dashboard** — Live metrics across all business modules
- 🚨 **AI Insights** — Auto-generated financial alerts by severity
- 🏥 **Health Score** — 0-100 composite financial health score
- 🤖 **AI Assistant** — Chat with your live financial data
- 📋 **Financial Report** — One-click executive summary report

## Modules Integrated
Accounting, Invoicing, CRM, HR, Expenses, Purchase, Project

## Installation
1. Copy `smartfinance_ai` folder into your Odoo addons path
2. Restart Odoo server
3. Go to Apps → Update App List → Search "SmartFinance AI" → Install

## Tech Stack
- Python (Odoo ORM)
- Odoo 19 Community Edition
- XML Views
- PostgreSQL
